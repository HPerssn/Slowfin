# Slowfin

A minimal full-stack portfolio tracking dashboard for long-term (“slow”) investors.  
Focused on clear allocation, performance, and projections rather than trading features.

**Live:** https://slowfin.hpersson.com  
**Project page:** https://www.hpersson.com/projects/slowfin <- More info in regard to process here!

---

## Features

- Portfolio value and allocation overview  
- Historical performance charts  
- Simple future performance simulations  
- Clean, low-noise UI, drawing inspiration from dieter rams philosophy

---

## Tech Stack

- **Backend:** ASP.NET Core  
- **Database:** SQLite (EF Core)  
- **Frontend:** Vue 3 + TypeScript  
- **Charts:** Chart.js  
- **CI/CD:** GitHub Actions  
- **Hosting:** Azure (API), Vercel (frontend)
